## Open Source
N8ech believes that patents are for losers and that open, maintainable, user-respecting hardware and software are a good thing. As much of N8ech's hardware and software products as possible are made with open source software, with open schematics and code. Anyone is free to use our designs and code for their own personal use. N8ech is still deciding on how to handle attribution and commercial use.
